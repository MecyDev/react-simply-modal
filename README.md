# react-simply-modal

## Install

```bash
npm install --save react-simply-modal
```

## Usage

```jsx

import MyComponent from "react-simply-modal";

export const Example = () => {
  render() {
    return <MyComponent content="sucess!" close={true}/>;
  }
}
```
